阿里云三维可视化使用初体验

发表于 2018-09-28 | 分类于 物联网

本文主要的目标是使用阿里云的云产品 - 物联网套件三维可视化

开始

准备工作

  1. 进入下载页面下载页面，点击“模型编辑器下载”
  2. 安装模型编辑器下载
  3. 安装完毕，启动模型编辑器下载，启动后界面如下所示
  4. 选择新建场景，开启场景自建

建立模型

  1. 基础操作
    • 鼠标左键 - 旋转、拖动
    • 鼠标滚轮 - 放大、缩小
    • 鼠标右键 - 平移场景、取消选中状态
    • 鼠标左键双击 - 进入模型内部
    • 鼠标右键双击 - 退出模型内部
    • 键盘空格 - 2D/3D 切换
  2. 新建场景后，会看到这样的样子
  3. 删除人物，现在开始建设场景
  4. 建立好围墙、并选择“室外-大门”
  5. 开始建立建筑，选择“室内-建房”，在刚才建好的园区里放一个“自定义建筑”，放好之后双击进入建筑内部，象第四部一样，先画一个矩形墙，然后根据下图中的①②③④选择好建筑外立面，并在选中建筑物后浮出的菜单中，选择“编辑”调整建筑物模型的大小，如果忘记设置了，也可以通过①②两步，重新回到选择建筑物外立面的界面
  6. 摆放好各种物体，其中物联网设备，一定要从右侧边栏- 设备-IOT设备选取，此处的模型会在下一步和实际的物联网设备关联
  7. 导出建好的模型，并非保存，参考下图步骤

关联模型

  1. 回到阿里云控制台，选择创建场景，并上传刚才导出的zip包，如果没有产品，需要到左边栏“设备管理”里创建一些设备，产品需要选择高级版本
  2. 点击确定之后，进行设备关联

产生联动

  1. 使用物联网套件SDK，把真实设备接入，请参考下一篇文章
  2. 真实设备接入之后，即可和刚才建模中的模型进行互动
